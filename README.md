# 🏨 Previsão de Cancelamento de Reservas com Machine Learning

## 📌 Problema de Negócio
Cancelamentos de reservas impactam diretamente a receita de hotéis, gerando prejuízos e dificuldades no planejamento operacional.

Este projeto tem como objetivo prever quais reservas têm maior probabilidade de cancelamento, permitindo ações preventivas.

---

## 🎯 Objetivo
Desenvolver um modelo de Machine Learning capaz de prever cancelamentos de reservas com base em dados históricos.

---

## ⚙️ Metodologia
- CRISP-DM
- Pré-processamento de dados
- Engenharia de atributos
- Balanceamento com SMOTE
- Treinamento e comparação de modelos

---

## 🧠 Modelos Utilizados
- Random Forest
- Gradient Boosting
- Regressão Logística
- SVM
- Naive Bayes
- MLP (Rede Neural)

---

## 📊 Métricas Avaliadas
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

---

## 🔍 Tecnologias
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

---

## 🚀 Resultados
O modelo foi capaz de identificar padrões de cancelamento com boa capacidade preditiva, podendo ser utilizado como apoio à decisão para redução de perdas.
